Welcome to Drago's List Review Part 1!

Use this resource to help you review how to get elements of a list in Scheme using car and cdr.

Firstly, a brief introduction to Scheme and list...
- Scheme is a dialect of the language Lisp (list programming)
- In scheme, lists are denoted with a single quote followed by an opening parenthesis, the content of the list, and a closing parenthesis
- The quote in the beginning prevents Scheme from evaluating it with its default evaluator
- Empty lists are written as '()
- Each list has two parts: the part containing just the first element and the part containing the rest of the list
- The first part of the list is known as the car and is accessible through the car operator: (car <list>)
    - Performing the car operator on a list can result in one of the following:
       - The first element of the list, the list has to contain at least one element
       - An error, this is because the argument given is an empty list
- The second part of the list is known as the cdr and is accessible through the cdr operator: (cdr <list>)
    - Performing the cdr operator on a list can result in one of the following:
       - The rest of the list containing at least one element, this is when the list has at least two elements
       - A null list, this is when the list has only one element
       - An error, this is when the list is an empty list

Example:
(define l  '(1 2 3))
(car l) --> 1
(cdr l) --> (2 3)
